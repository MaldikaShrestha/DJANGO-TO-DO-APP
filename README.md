##DJANGO TO DO APP
- This is a simple task management web application built using Django 
- Users can create a task, update or delete them
- The project demonstrates CRUD operations using Django models, views, and templates.

##FEATURES
- Users can create a new task
- Users can update the existing task 
- Users can delete the task
- Users can view all the existings task
